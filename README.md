## Installation

```
npm i create-react-npm
```

## How to use

1. Create a new folder with the component to publish `your_react_component/src/componentToPublish.js` and then

```
cd your_react_component
```

2. Execute the command

```
creacte-react-npm
```

3. Ready to publish!

```
npm publish
```
